#### Why JEST?
All in one Framework
* test runner
* assertion library
* mocking
* code-coverage
* readable errors
* minimal config
* fast (parallelzation)
* IDE support (VS Code)

---


#### How to install

```
npm init
npm install --save-dev jest
npm install --save @types/jest
```
